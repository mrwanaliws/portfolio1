# Marwan Ali Mohamed — AI and Data Analysis Engineer

A premium AI and data portfolio built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

Key links

- GitHub: https://github.com/marwanali
- LinkedIn: https://www.linkedin.com/in/marwan-ali-mohamed

Installation

```bash
npm install
npm run dev
```

Build for production

```bash
npm run build
npm start
```

Deploy to Vercel: push to a Git repo and import the project in Vercel. Set environment variables from `.env.example` if using email or contact integrations.
